**Warung Fina Berry** is a modern Point of Sale (POS) and restaurant management application built with Flutter. It aims to streamline operations, including manual menu management, order processing, and administrative tasks, while featuring an intuitive user interface and a custom branding aesthetic.

## 🚀 Features

- **Menu Management**: Easily add, edit, and categorize menu items (e.g., Main Course, Dessert, Drinks).
- **Image Uploads**: Manual file-based upload system for menu item images using local device storage.
- **Admin Dashboard**: Comprehensive dashboard for cashiers and administrators to manage orders, inventory, and application settings.
- **RESTful API Integration**: Connected to a robust Laravel backend for managing data synchronization and seamless operations.
- **Cross-Platform**: Built on Flutter to provide a smooth, native-like experience on both Android and iOS devices.

## 🛠️ Tech Stack

- **Frontend**: [Flutter](https://flutter.dev/) (Dart)
- **Backend**: Laravel (PHP) + MySQL Database
- **Packages/Dependencies**:
  - `image_picker` for manual menu photo uploads.
  - HTTP packages for API connections.

## ⚙️ Getting Started

### Prerequisites

- Flutter SDK (latest stable version)
- Dart SDK
- Laravel Backend setup & running locally (port 8000)

### Installation

1. **Clone the repository:**
   ```bash
   git clone <repository_url>
   cd flutter_application_finna_berry
   ```

2. **Install dependencies:**
   ```bash
   flutter pub get
   ```

3. **Run the backend server:**
   Make sure your Laravel backend is running. You can use the provided startup scripts (`start_project.bat` / `start_project.ps1`) to launch your development environment quickly.

4. **Run the app:**
   ```bash
   flutter run
   ```
